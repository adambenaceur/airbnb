# search functionality

```React-Date-Range```: A date library agnostic React component for choosing dates and date ranges. Uses ```date-fns``` for date operations.

# map function
Map-box : location data api tool to locate air-bnb locations using (longitude, latitude) 
current function is only visuals. Future iteration should look at implementation of dynamic datasets into map-box api
